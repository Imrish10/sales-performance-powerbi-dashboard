# Sales Performance Analysis Dashboard (Power BI)

## Project Overview
This project analyzes sales performance using Power BI dashboards.

## Key Metrics
- Total Revenue
- Total Profit
- Units Sold
- Average Order Value

## Dashboard Insights
• Bangalore generated the highest revenue  
• Electronics category dominates sales  
• Sales trend declines in later months  

## Features
- Interactive filters for city, category, and month
- Sales trend visualization
- Category performance comparison

## Tools Used
Power BI, Data Visualization
